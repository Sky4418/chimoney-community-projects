# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2025-03-03

### Added
- Initial release of n8n-nodes-chimoney
- Payout resource with operations:
  - Initiate payout (Chimoney, Bank Transfer, Mobile Money, Airtime, Gift Card)
  - Get payout status
- Account resource with operations:
  - Create virtual account
- Wallet resource with operations:
  - Get wallet balance
- API Key authentication support
- Sandbox and Production environment support
